# Print the Pattern

![Pattern #3](https://github.com/Swapnadip2005/Java_DSA_Insider/assets/149895037/5f6f9f79-56e9-4f13-a9f9-d9cf69c489a4)

```java
public class Pattern3 {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            for (int j = 1; j <= i ; j++) {
                System.out.print(j);
            }
            System.out.println();
        }
    }
}
```
