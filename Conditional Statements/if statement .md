# if Statement 
The `if statement` is used to execute a block of code only if a specified condition evaluates to `true`.
### Syntax
```java
if(condition) {
  //code
}
```
# Example
```java
int x = 20;

if (x <= 25) {
  System.out.println(x + " is less than 25"); // Output = 20 is less than 25
}
```
