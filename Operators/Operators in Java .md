# Operators

`Operators` are special symbols or keywords that perform operations on `Operands`

Sum = a + b

- '+' is a `Operator`
- 'a' & 'b' are `Operands`

# Types of Operators

- Arithmetic Operators
   - Binary Operators
   - Unary Operators
- Relational Operators
- Logical Operators
- Bitwise Operators (Bit Manipulation)
- Assignment Operators
